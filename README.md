# BitNFC_reactive

### BitNFC inesita fork - the original and working app is here: https://github.com/720kb/BitNFC

this is just an attempt

## Components

- bitcore js library
- coffeescript code taken from blockchain pen for the transactions and network code
- inesita for the ui


## Installing

    npm install bower -g


    bower install

## Development

    gem i inesita


    inesita server

## Structure

- account
  - name
  - public address
  - qrcode
  - settings

- balance (opzione hide / pin required)


- receive
- send

- transactions
